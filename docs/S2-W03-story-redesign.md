# S2-W03《消失的慶典皇冠》— 故事＋機制重寫版

> 定稿方向：長度控制＝**真的在審問關係人**（請他「說詳細一點／說簡潔一點」，只有一邊吐出關鍵字）。
> W01（三要素）、W02（格式）**點到為止**複習。左側**關鍵字欄**全程累積。結尾**真相重組頁**：把關鍵字拖進空格、親手組出真相。
> 主題昇華：**話說得太短，好人也會被當成小偷。** 長度是這樁案子被誤解的原因。

---

## 全案流程

```
開場劇情（打字＋阿問配音，每段 gate 檢核）
  → 第1章 → 第2章 → 第3章 → 第4章 → 第5章
  → 真相重組頁（拖關鍵字入空格）→ 結局＋TA
左側「關鍵字欄」從第1章開始累積，一章長大一點，最後餵給真相重組頁。
```

**每章迴圈**
1. 抵達現場・案情（一段敘事）
2. **蒐證**：在場景圖上點熱點找線索 → 每條線索 = 一個**關鍵字**飛進左欄
3. **審問關係人**（長度控制核心）：關係人先給一段話（有的落落長、有的太短）→ 你請他「**說簡潔一點**」或「**說詳細一點**」→ **只有對的那一邊**吐出關鍵資訊 → 又一個關鍵字飛進左欄
4. 破章 → 下一章
- （第2章順路複習 W01 三要素、第4章順路複習 W02 格式，見各章）

## 第一章・落落長的報案　（長度：落落長 → 請他「簡潔」）

**案情**
> 夜裡的展示廳好安靜，安靜到你聽得見自己的腳步聲。
> 明天早上，全鎮的人都會擠進來看那頂**慶典皇冠**——金色的，正中央一顆藍寶石，亮得像把整個夜空縮成一顆星。
> 可是現在，玻璃櫃的門開著。紅色的軟墊上，空空的。**皇冠不見了。**
> 阿問閃了一下燈：「嗶嗶。天一亮典禮就開始，我們只有今晚。第一個要問的人，就在你面前發抖。」

**蒐證（點熱點）**
- 展示櫃 → 「櫃門沒被撬開，是用鑰匙正常打開的。」→ 關鍵字【沒被撬】
- 牆上的鐘 → 「停在九點，是巡邏發現的時間。」→（干擾）
- 地板 → 「有一點白白的粉末。」→（干擾，之後才知道是糖霜）

**審問・老陳（糊塗管理員）**
他一開口就停不下來：
> 「唉喲你不知道，我今天六點就來了，孫子昨天發燒我沒睡好，膝蓋又痛，晚餐吃滷肉飯配湯，我還泡了茶擦了玻璃……」

你請他「**說簡潔一點**」 → 「**皇冠是八點到九點之間不見的。**」→ 關鍵字【8:00–9:00】
（若請他「說詳細一點」→ 又是一串廢話，重點更埋。**這就是太長的壞處：重點被淹沒。**）

抓到重點後，一句話尾巴帶出破口：「櫃門沒被撬……那開櫃的人有鑰匙。」老陳臉一白：「備份鑰匙……不見了！」→ 關鍵字【備份鑰匙】
→ **破章 / 逼出下一章：去後台，看誰動了鑰匙箱。**

## 第二章・閃躲的記者　（長度：太短 → 請她「詳細」）（順路複習 W01 三要素）

**案情**
> 後台堆滿道具，燈只開一半。一個穿深藍外套的人影，一看到你就把臉別開——是她，鬼鬼祟祟的小葉。

**蒐證**
- 鑰匙箱 → 「開著的，備份鑰匙真的不見了。」→（確認）
- 衣架外套口袋 → 「一支錄音筆。」→（干擾）

**複習 W01：問清楚才問得到**
小葉很防備。你要先把問題**組清楚**才問得動她——
選對【對象：小葉】＋【根據：你 8:35 在後台】＋【要問：你看到了什麼】。
（問得不清楚 → 她只回「不知道」。）

**審問・小葉（記者）**
她只肯給**太短**的回答：
> 「我……沒幹嘛。就路過。」

你請她「**說詳細一點**」 → 「好啦！我是記者，想偷拍明天的皇冠。我八點三十五在後台，**手機拍到一個紅色的影子，抱著東西、小心翼翼往祈福亭去**——不是我！秀琴阿姨可以證明我一直在這。」
→ 關鍵字【紅色影子】、【祈福亭】；小葉**洗清**。
（若請她「說簡潔一點」→ 「就……有個紅影子。」講不清，沒用。）
→ **破章 / 逼出下一章：紅影子把東西帶去祈福亭了。**

## 第三章・只寫得下四個字的祈福牌　（蒐證重心；審問阿福：落落長 → 簡潔）

**案情**
> 祈福亭掛滿小小的祈福牌，風一吹叮叮響。慶典的習俗——每塊牌子只寫得下四個字。紅影子在這停過。

**蒐證**
- 祈福牌（只有四格）→ 上面寫著「**修好就還**」→ 關鍵字【修好就還】
- 亭子角落 → 「一隻白手套。」→ 關鍵字【白手套】

**審問・阿福（點心師傅）**
手套是他的，他一緊張就講不停：
> 「那手套是我的沒錯啦！可是我今天做了三層蛋糕，先打鮮奶油再調糖霜，跟三個廚工聊我兒子考試，我整晚都在廚房沒出去啊……手套我都放後台架上，誰都拿得到！」

你請他「**說簡潔一點**」 → 「**手套上的糖霜，是放在後台架子上沾到的，不是我戴的時候弄的。**」→ 關鍵字【糖霜】
→ 手套沾糖霜、又是阿福的 → **直覺指向阿福**（誤導）。但他有三個廚工作證。
→ **破章 / 逼出下一章：那手套到底是誰戴走的？兩份說法對不上。**

---

## 第四章・對不上的兩份說法　（長度：追問；順路複習 W02 格式＝時間軸）

**案情**
> 阿福喊冤、秀琴堅持看到紅影子。兩邊兜不起來。阿問說：「嗶——這種『誰在哪、幾點』的亂帳，要**排成時間軸**才看得清。」（← W02 格式複習：步驟/時間軸）

**審問・秀琴（打掃阿姨）**
她講得**太短**：
> 「我就……看到紅紅的一個影子。」

你請她「**說詳細一點**」 → 「八點四十五我在補衛生紙，看到一個**紅色的影子**從後台把手套拿走，往大廳去。那時候廚房、走廊都有人，只有辦公室那邊……安安靜靜的。」

**複習 W02：排時間軸**
把四個人拖上 8:00–9:00 的時間軸（＝W02 的步驟/格式）：
- 阿福 廚房（三廚工證）、秀琴 走廊→廁所、小葉 後台（洗清）
- **8:40–8:55 只有林主席說在辦公室打電話——但總機查無紀錄** → 關鍵字【8:40–8:55 沒證明】
- 深紅套裝＝紅色影子、備份鑰匙她保管 → 關鍵字【林主席】
→ 此刻**故意讓玩家認定「主席是小偷」**。
→ **破章 / 逼出下一章：去找林主席當面對質。**

---

## 第五章・最後一問（翻轉）　（長度：全自主；問對才翻轉）

**案情**
> 你站到林主席面前。她穿著深紅色套裝，手裡緊緊抱著什麼。空氣好安靜。

**審問・林主席**——你自己決定要問得多長。
問「妳做了什麼？」請她「簡潔」 → 「……是我拿走皇冠的。」→（確認，但像認罪）
問「為什麼？」——
- 若請她「**簡潔**」 → 「因為寶石鬆了。」聽起來像藉口。
- 你改請她「**說詳細一點**」 →
> 「今天傍晚巡場，我發現皇冠正中央那顆藍寶石——**爪座鬆了**。明天只要有人戴著它走上台、輕輕一晃，寶石就會當場掉下來摔碎。我想今晚偷偷送去修、**天亮前放回來**，就沒人會知道、慶典也能平平安安。我沒說，是因為……它是在我任內鬆掉的，我怕大家怪我。」
→ 關鍵字【爪座鬆了】、【天亮前修好放回】。
（**這一問就是翻轉**：太短＝像藉口，詳細＝才聽見她的真心。）

你忽然想起——祈福牌「修好就還」、照片裡她**抱著**皇冠小心翼翼的。
**她從第一步就在說「我會還」，只是話太短，你差點把恩人當小偷。** → 關鍵字【太短】

## 真相重組頁（全案結尾）

一整頁真相文本，中間留空格；左欄所有關鍵字（含干擾）供你拖入。

> 那一夜，慶典皇冠從展示櫃消失了。
> 櫃門〔沒被撬〕，是有人用〔備份鑰匙〕打開的——會用鑰匙的，是有權限的人。
>
> 排出時間軸才發現：〔8:40–8:55 沒證明〕這段，只有一個人說不出自己在哪——〔林主席〕。
> 秀琴看到的〔紅色影子〕，是她的深紅套裝；她戴上〔白手套〕怕留指紋，手套才沾了〔糖霜〕。
>
> 可是，她不是小偷。
> 她發現皇冠的寶石〔爪座鬆了〕，怕明天摔碎，想〔天亮前修好放回〕。
> 她把「〔修好就還〕」寫在祈福牌上——她一直在說她會還，
> 只是那句話〔太短〕了，短到我們差點錯怪了一個好人。

- **空格 11 個**，答案來自左欄關鍵字；左欄另有 3 個干擾字（停在9點的鐘、錄音筆、白粉末）不該填進去（＝W02「挑重點」複習）。
- 全對 → 破案動畫＋翻轉台詞＋金幣＋TA。

**結局**
> 天亮前，老鐘錶行把修好的皇冠送了回來，寶石牢牢的。慶典照常，林主席跟大家道歉，大家反而謝謝她。
> 阿問：「嗶。你破的這個案子，最重要的不是抓到誰——是你終於**聽懂了**一個話沒說完的人，心裡真正的意思。」

## 給工程的機制清單（引擎沿用，只加/改教學層）
1. **左側關鍵字欄**：常駐面板，`S.keywords[]` 累積；找到線索/審問成功時 push 並飛入動畫。
2. **審問環節**：每章 `interview`：關係人頭像＋初始台詞（太長/太短）＋兩顆按鈕「說簡潔／說詳細」；對的一邊 → 揭露台詞＋掉關鍵字；錯的一邊 → 軟提示、可重試（防硬 A：非致命）。
3. **W01 複習（2章）**：問話前組「對象＋根據＋問題」三要素，齊了才問得動。
4. **W02 複習（4章）**：時間軸拖放（沿用現有 timeline），框成「格式＝步驟」。
5. **真相重組頁**：整頁文本挖空格；左欄關鍵字 chip 拖入；含干擾字；全對破案。
6. 金幣 60×5＋破案 60＝360；存檔綁 uid；純前端本地判定＋預寫。
